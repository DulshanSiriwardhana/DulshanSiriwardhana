## 🏆 Masterpiece Project Showcase

### KnightChess: Real-Time Decentralized Multiplayer Platform
*(The complete source code and interactive architecture for my proudest build)*

<br />

#### 📌 System Overview
KnightChess is a premium, highly scalable online chess application built from the ground up to rival industry standards like Chess.com. It is orchestrated on a modern Microservices architecture, providing immediate, zero-latency feedback during live competitive matches globally.

<br />

#### ⚙️ Core Architecture & Flow Design

1. **The Frontend Experience:**  
   Developed using **React, Vite, and TypeScript**, the interface is incredibly fast and strongly typed to prevent runtime state errors during matches. The UI represents a masterclass in modern web design utilizing **Tailwind CSS 4.0**. I incorporated custom spectral fonts, dynamic glass-morphism execution panels, and highly polished scrollbars to deliver a breathtaking digital aesthetic.

2. **The Real-Time Data Gateway:**  
   Traditional HTTP requests were far too slow for competitive speed chess. I engineered a dedicated, decoupled **Socket.IO** bidirectional pipeline which broadcasts complex board-state array transformations across the WebSocket protocol. This allowed me to achieve near-instantaneous global synchronization between clients.

3. **Backend Logistics & Authentication:**  
   The primary application server runs on **Node.js** utilizing an **Express.js** gateway API. This backend features a heavily secured **JWT-based authentication** service architecture. All persistent match histories, user profiles, securely hashed passwords, and competitive Elo algorithmic ratings are heavily encrypted and safely stored using **MongoDB**.

<br />

#### 🚀 Key Features Implemented

- **Advanced Distributed State Management:** Custom engineered a centralized functional store to instantly reflect and validate complex standard chess mechanics (En Passant, Castling, Pawn Promotion, Check/Checkmate traversal logic).
- **Matchmaking & Global Lobby System:** Built an interactive, live lobby structure allowing thousands of users to concurrently broadcast public challenges, share match URLs, or dynamically join active open game rooms seamlessly.
- **Robust Security Principles:** Protected sensitive internal API endpoints via rigorous CORS configuration profiling, targeted rate-limiting protocols, and strict validation of inputs to completely prevent NoSQL injection vectors.
- **Custom Open Source Node Library:** Personally contributed to the overarching open-source community ecosystem by refining and utilizing a highly accurate, pure functional React state generation library.

<br />

#### 🧠 Engineering Impact & Methodological Learnings
Designing and deploying the KnightChess project forced a deep, hands-on understanding of concurrent connection scaling algorithms. Having to personally handle the memory footprint optimization strategies across thousands of actively open concurrent web-sockets taught me advanced Linux deployment optimizations and container memory management workflows. It stands today as my absolute best representation of a responsive, full-stack, distributed web application.
