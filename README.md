## 4. Featured Engineering Projects

Below is a curated selection of architectural projects demonstrating my capacity to handle diverse software requirements from conception to deployment.

### 4.1 Enterprise E-Commerce Data Analytics Pipeline
A comprehensive Big Data pipeline engineered to capture, process, and analyze high-velocity clickstream data in real time using a Kappa Architecture framework.
- **Architectural Flow:** Leveraged Apache Kafka as the central ingestion nervous system to handle highly concurrent event streams from E-Commerce mock users.
- **Processing Layer:** Implemented Apache Spark Structured Streaming to apply real-time analytical queries and metric aggregations. 
- **Storage & Orchestration:** Integrated PostgreSQL for robust, transactional data storage and orchestrated complex data lifecycle dependencies utilizing Apache Airflow scheduling mechanisms.
- **Impact:** Demonstrated ability to construct robust, enterprise-grade ingestion systems that provide immediate strategic business intel.

### 4.2 Decentralized Fund Release Protocol (Web3)
A robust smart contract and distributed web application ecosystem acting as a democratic crowdfunding incubator.
- **Smart Contract Governance:** Engineered a highly secure "Request-Vote-Finalize" decentralized governance model using Solidity. This ensures project creators can only withdraw project funds if milestones are democratically approved by contributing investors.
- **Decentralized Storage:** Migrated centralized image hosting to IPFS via Pinata to establish an immutable multimedia history.
- **Wallet Infrastructure:** Built resilient frontend integrations supporting Ethers.js. Created dynamic fallback strategies utilizing generic WalletConnect integration alongside standard browser-extension MetaMask support.
- **Impact:** Showcased proficiency in combining traditional MERN stack patterns with immutable blockchain data layers safely.

### 4.3 Real-Time Distributed Multiplayer System (KnightChess)
A fully functional, real-time chess platform rivaling industry leaders in state execution speed.
- **Systems Architecture:** Built upon a decoupled microservices architecture utilizing Node.js and Express behind a secure gateway.
- **Real-Time Execution:** Replaced traditional polling mechanics with highly optimized bidirectional WebSockets (Socket.IO) resulting in near-zero latency state updates between players.
- **Frontend Systems:** Crafted using strictly typed React (Vite+TS) and tailored UI/UX elements compiled through Tailwind CSS to deliver an uncompromised competitive gaming experience.
- **Impact:** Refined algorithmic efficiency handling complex game states and concurrent users simultaneously.
